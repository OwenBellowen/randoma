# randoma
An NPM package that contains random functions to help you in projects and etc.

## Installation
```bash
npm install randoma
```

## Features
- Generators
  - Random Number
  - Random String
  - Random Color
  - Random Hex
  - Random RGB
  - Random RGBA
  - Random HSL
  - Random HSLA
- Converters
  - RGB to Hex
  - Hex to RGB
  - RGB to HSL
  - HSL to RGB
  - RGB to RGBA
  - RGBA to RGB
  - HSL to HSLA
  - HSLA to HSL
- Validators
    - Is Hex
    - Is RGB
    - Is RGBA
    - Is HSL
    - Is HSLA
- Collection Util
- Logging Util
- Encryption Util
- Date Util
- Array Util
- String Util
